# 채용광고 썸네일 시스템 기획 및 액션플랜

## 1. 기획 배경 및 목적

### 문제점 (경쟁사 분석)
- 퀸알바 등 경쟁 사이트에서는 기업회원이 **직접 GIF 파일을 준비**하여 썸네일로 사용
- GIF를 준비하지 못하는 회원은 **상대적 박탈감**을 느낌
- 각양각색의 GIF가 난립하면서 **사이트 전체가 지저분**해지는 문제
- 디자인 통일성이 없어 사이트 신뢰도 하락

### 이브알바 해결 방향
- **미리 디자인된 그라데이션 컬러 테마 20종**을 제공
- 기업회원이 컬러 선택 + 문구 작성만으로 **깔끔한 썸네일 생성**
- 사이트 전체의 **디자인 통일성 유지** + 회원 간 **공평한 노출 품질**
- CSS 기반 렌더링으로 **이미지 파일 불필요**, 실시간 수정 반영

## 2. 썸네일 기본 구성

### 썸네일 카드 구조
```
┌─────────────────────────────────┐
│  [아이콘 뱃지] ← 유료옵션      │ ← 그라데이션 배경
│                                 │
│     업소명 / 타이틀 (큰 글씨)    │ ← 흰색 텍스트
│     홍보 문구 (1~2줄)           │ ← 흰색 또는 반투명 텍스트
│                                 │
├─────────────────────────────────┤
│ 지역 | 업종 | 급여 | 조회/일수   │ ← 하단 정보 (카드 밖)
└─────────────────────────────────┘
```

### 기업회원 입력 항목
| 항목 | 설명 | 글자 수 제한 |
|------|------|-------------|
| 컬러 테마 | 20종 중 1개 선택 (필수) | - |
| 타이틀 | 업소명 또는 대표 문구 (기본값: 닉네임/상호) | 15자 이내 |
| 홍보 문구 | 어필 포인트 1~2줄 | 40자 이내 |

### 썸네일 노출 위치
| 위치 | 크기 | 광고 등급 |
|------|------|----------|
| 메인 우대채용정보 | 큰 카드 (2열) | 우대 이상 |
| 메인 프리미엄채용정보 | 중간 카드 (5열) | 프리미엄 |
| 메인 스페셜/추천 | 작은 카드 | 스페셜/추천 |
| 채용정보 리스트 | 리스트 좌측 아이콘 | 전체 (줄광고/급구 제외) |
| 지역별 채용정보 | 카드형 | 전체 (줄광고/급구 제외) |

**제외**: 줄광고, 급구채용은 텍스트 리스트 형태이므로 썸네일 미적용

## 3. 컬러 테마 프리셋 (20종)

### 설계 원칙
- 각 컬러는 **2~3색 그라데이션** (단색 금지)
- 위에서 흰색 텍스트가 잘 보이도록 **충분한 명도 대비** 확보
- 서로 확연히 구분되는 색상 배치 (유사색 최소화)
- 카테고리: 따뜻한 톤 / 차가운 톤 / 고급 톤 / 비비드 톤 / 다크 톤

## 4. 유료 옵션 (프리미엄 효과)

### 기존 유료 옵션 (현재 구현됨)
| 옵션 | 설명 | 가격 |
|------|------|------|
| 아이콘 추가 | 썸네일 우측 상단 뱃지 (초보환영, 원룸제공 등 9종) | 30일 3만 / 60일 5.5만 / 90일 7만 |
| 형광펜 | 채용정보 리스트 제목 배경색 (8색) | 30일 3만 / 60일 5.5만 / 90일 7만 |
| 굵은글씨 | 채용정보 리스트 제목 볼드 표시 | 30일 3만 / 60일 5.5만 / 90일 7만 |

### 신규 유료 옵션 (제안)

#### 옵션 A: ✨ 글자 반짝임 효과
- **설명**: 썸네일의 타이틀/홍보 문구에 CSS 반짝임(shimmer) 애니메이션 적용
- **효과**: 텍스트 위를 빛이 지나가는 효과 (고급스러운 느낌)
- **구현**: CSS `@keyframes shimmer` + `background-clip: text` + `linear-gradient` 애니메이션
- **적용 범위**: 썸네일 카드의 타이틀 텍스트
- **가격 제안**: 30일 4만 / 60일 7만 / 90일 9만 (기존 옵션보다 소폭 높게)

#### 옵션 B: 🌊 웨이브 컬러 효과
- **설명**: 썸네일 그라데이션 배경이 물결치듯 움직이는 애니메이션 (현재 메인 헤더와 유사)
- **효과**: `background-size: 200% 200%` + `@keyframes wave` 로 그라데이션 흐름
- **구현**: CSS 애니메이션만으로 가능 (JS 불필요, 성능 부담 최소)
- **적용 범위**: 썸네일 그라데이션 배경 전체
- **가격 제안**: 30일 5만 / 60일 9만 / 90일 12만 (가장 눈에 띄는 효과)

#### 옵션 C: 💎 프리미엄 테두리 효과
- **설명**: 썸네일 카드에 금색/은색 테두리 + 미세한 발광 효과
- **효과**: `box-shadow` + `border` 애니메이션 (은은하게 발광)
- **구현**: CSS `@keyframes glow` + `box-shadow` 변화
- **적용 범위**: 썸네일 카드 외곽
- **가격 제안**: 30일 3만 / 60일 5.5만 / 90일 7만 (기존 옵션과 동일)

#### 옵션 D: 🔥 HOT/NEW 뱃지
- **설명**: 썸네일 우측 상단에 "HOT", "NEW", "추천" 등 강조 뱃지 (현재 이브알바에도 일부 존재)
- **효과**: 빨강/초록 등 눈에 띄는 색상의 작은 뱃지
- **구현**: CSS 절대 위치 + 뱃지 스타일
- **적용 범위**: 기존 아이콘 추가와 별개로 운영 가능
- **가격 제안**: 기존 아이콘 추가에 포함 또는 별도 상품

## 5. 구현 범위 및 단계

### 수정 대상 파일 (예상)

| 파일 | 변경 내용 |
|------|----------|
| `jobs_register_main.php` | 광고유료결제 섹션에 "썸네일 설정" UI 추가 |
| `jobs_view_main.php` | 에디터에 "썸네일 설정" 섹션 추가 (컬러 선택, 문구 입력, 미리보기) |
| `jobs_register_update.php` | 썸네일 데이터 저장 (jr_data에 thumb_* 키) |
| `jobs_basic_info_save.php` | 에디터에서 썸네일 수정 저장 |
| `index_main.php` | 메인 페이지 썸네일 렌더링 |
| `jobs_ongoing_main.php` | 채용정보 리스트 썸네일 렌더링 |
| 지역별 채용 PHP | 지역별 채용 썸네일 렌더링 |
| `evealba.css` 또는 별도 CSS | 썸네일 스타일, 애니메이션 효과 정의 |
| 별도 config (extend/) | 컬러 프리셋 20종 정의 |

### 구현 단계

#### Phase 1: 기본 썸네일 시스템 (핵심)
1. 컬러 프리셋 20종 정의 (`extend/thumbnail_config.php`)
2. `jobs_view_main.php`에 썸네일 설정 섹션 추가
   - 컬러 선택 그리드 (20종)
   - 타이틀 / 홍보 문구 입력
   - 실시간 미리보기 카드
3. 저장 AJAX (`jr_data`에 `thumb_gradient`, `thumb_title`, `thumb_text` 저장)
4. `jobs_register_main.php`에도 동일 UI 추가

#### Phase 2: 노출 연동
5. 메인 페이지 (우대/프리미엄/스페셜) 썸네일 렌더링
6. 채용정보 리스트 썸네일 렌더링
7. 지역별 채용정보 썸네일 렌더링

#### Phase 3: 유료 효과 옵션
8. 글자 반짝임 효과 (CSS 애니메이션)
9. 웨이브 컬러 효과 (CSS 애니메이션)
10. 프리미엄 테두리 효과 (CSS 애니메이션)
11. 결제 연동 (기간별 가격, 결제 시 효과 활성화)

## 7. 주의사항 및 고려사항

1. **성능**: CSS 애니메이션은 GPU 가속을 사용하므로 성능 부담 최소. 다만 한 페이지에 다수의 애니메이션 카드가 동시 노출될 경우 모바일에서 테스트 필요
2. **접근성**: 반짝임/웨이브 효과는 `prefers-reduced-motion` 미디어 쿼리로 사용자 설정 존중
3. **모바일 대응**: 썸네일 카드가 모바일에서도 깔끔하게 표시되도록 반응형 설계
4. **기존 데이터 호환**: 기존 회원의 jr_data에 thumb_* 키가 없는 경우 기본값(첫 번째 컬러 + 닉네임) 적용
5. **미결제 회원**: 기본 컬러 테마 선택은 무료, 효과 옵션만 유료
6. **아이콘 뱃지 연동**: 기존 "아이콘 추가" 결제 시 썸네일 우측 상단에 해당 뱃지 표시

## 8. 동일한 실수 방지

| 실수 유형 | 예방책 |
|----------|--------|
| 컬러 프리셋 CSS가 브라우저마다 다르게 보임 | Chrome/Safari/Firefox/모바일 크로스 브라우저 테스트 |
| jr_data에 새 키 추가 시 기존 데이터 손상 | JSON merge 방식 사용, 기존 키 덮어쓰기 금지 |
| 유료 효과 만료 후에도 계속 표시됨 | 렌더링 시 expire 날짜 체크 로직 필수 |
| 썸네일 문구에 XSS 공격 코드 입력 | htmlspecialchars() 적용 필수 |
| 메인/리스트/지역별 3곳의 렌더링 불일치 | 공통 렌더링 함수 1개로 통일 |
