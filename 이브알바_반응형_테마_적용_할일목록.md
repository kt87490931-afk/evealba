### 1.1 반응형 구성

| 브레이크포인트 | 적용 내용 |

| **태블릿 (≤900px)** | 1컬럼, 좌측 사이드바 숨김, 모바일 추천업소 표시, 우대채용 2열, 퀵스탯 3열, top-bar 좌측·카카오 버튼 숨김, 네비 터치 스크롤 |

- **좌측 사이드바**: 로그인(상단) → 빠른메뉴 → 지역별검색 → 추천업소 (플로팅 아님)

- **플로팅 CTA**: 카카오톡, 채팅, 상단이동
- **채팅 패널**: 우측 슬라이드 패널

## 4. 동일 실수 방지용 해결책

| 상황 | 해결책 |
|------|--------|
| PC/모바일 CSS 충돌 | evealba 전용 클래스는 `ev-` prefix 사용 고려, 기존 `#hd`, `#ft` 등 id는 점진적 교체 |
| outlogin 스킨 불일치 | `outlogin('theme/xxx')` 스킨을 evealba 마크업에 맞게 신규 제작 또는 기존 스킨 수정 |
| 채팅 패널이 안 열림 | `chat_box.php` 의존성(DB, config) 확인, 연동 전에는 placeholder만 표시 |
| 반응형이 적용 안 됨 | viewport 메타 태그, CSS 미디어쿼리 로드 순서 확인 |
| 좌측 사이드바가 플로팅처럼 보임 | `position:sticky` 제거 여부 재확인 (현재 HTML에는 없음) |
