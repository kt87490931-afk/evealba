# 채용광고 자동배포 시스템 액션플랜

## 현재 상태 분석

### 현재 흐름 (수동)
```
[기업회원 등록] → pending, payment_confirmed=0, approved=0
       ↓
[관리자 입금확인] → payment_confirmed=1  (AI큐 등록)
       ↓
[관리자 승인] → approved=1, status='ongoing', end_date 설정
       ↓
[기업회원이 직접] → 썸네일 생성 → 광고글 확인 → "광고글 배포하기" 클릭
       ↓
일반유저에게 노출 (← 공개 페이지가 아직 정적 HTML이라 실제 노출 불가)
```

### 핵심 문제점
1. 공개 페이지(메인/채용정보/지역별)가 DB와 연결되지 않음 (정적 HTML 더미 데이터)
2. 기업회원이 "배포하기"를 안 누르면 입금 후에도 광고 미노출
3. 등록 시 기본 썸네일이 없어서 배포해도 빈 카드 표시
4. 입금확인 + 승인이 분리되어 불필요한 2단계 절차

---

## 목표 흐름 (자동배포)

```
[기업회원 등록]
  → pending, payment_confirmed=0
  → [자동] 기본 썸네일 생성 (무료20종 랜덤 + 닉네임/업소명 + "안녕하세요")
       ↓
[관리자 입금확인 1클릭]
  → payment_confirmed=1, approved=1, status='ongoing', end_date 설정
  → [자동] AI 소개글 큐 등록
  → [자동] 결제한 광고 상품 영역에 즉시 노출 시작
       ↓
[기업회원 선택사항] → jobs_view.php에서 썸네일 커스터마이징 (유료옵션 추가)
```

## 단계별 액션플랜

### Phase 1: 기본 썸네일 자동 생성 (등록 시점) — 15분

**대상 파일**: `jobs_register_update.php`

INSERT 전에 jr_data JSON에 기본 썸네일 데이터 자동 삽입:
- 무료 20종 중 랜덤 컬러 → `thumb_gradient`
- 닉네임 또는 업소명 → `thumb_title`
- "안녕하세요" → `thumb_text`
- 기본값: 흰색 텍스트, 아이콘/모션/웨이브/테두리 없음

---

### Phase 2: 입금확인 = 자동 승인+배포 (관리자 1클릭) — 30분

**대상 파일**: `adm/jobs_register_confirm_update.php`

입금확인 시 자동으로 승인까지 한번에 처리:
- `jr_payment_confirmed = 1`
- `jr_approved = 1`
- `jr_status = 'ongoing'`
- `jr_approved_datetime = 현재시각`
- `jr_end_date = 현재일 + jr_ad_period(일)`
- AI 소개글 큐 등록 (기존 로직 유지)

`approve_update.php`는 삭제하지 않고 유지 (예외 케이스용)

---

## 실패 시 해결책

### 썸네일 자동 생성 시 빈 값
- **원인**: jr_data JSON에 필수 키 누락
- **해결**: 등록 시 기본값 검증, NULL/빈문자열 → 기본값 대체

### 입금확인 후 자동 승인이 안 됨
- **원인**: confirm_update.php에서 SQL 오류
- **해결**: 실패 시 관리자에게 에러 메시지 표시, approve_update.php로 수동 처리 가능

### 공개 페이지에서 광고가 안 보임
- **원인**: jr_ad_labels 파싱 오류 또는 jr_status 조건 불일치
- **해결**: 관리자 목록에서 해당 광고의 status/labels 직접 확인

### 광고 기간 만료 후에도 노출됨
- **원인**: jr_end_date 체크 누락
- **해결**: 쿼리에서 `jr_end_date >= CURDATE()` 조건 필수 적용
- 추후: 크론잡으로 매일 만료 광고 `status='ended'` 처리 (`cron/jobs_expire_check.php`)

### 서버 배포 후 반영 안 됨
- **원인**: 서버 경로 `/var/www/evealba`, git remote HTTPS
- **해결**: `ssh root@188.166.179.115 "cd /var/www/evealba && git fetch origin && git reset --hard origin/main"`

## 참고: DB 테이블

### g5_jobs_register (기존 컬럼)
| 컬럼 | 설명 |
|---|---|
| jr_id | PK, 자동증가 |
| mb_id | 회원 ID |
| jr_status | pending / ongoing / ended |
| jr_payment_confirmed | 0=대기, 1=확인 |
| jr_approved | 0=대기, 1=승인 |
| jr_approved_datetime | 승인일시 (광고시작일) |
| jr_data | JSON (썸네일+폼+AI콘텐츠 전부 포함) |
| jr_ad_labels | "줄광고 30일, 우대 60일" 형태 |
| jr_ad_period | 광고기간 (일수) |
| jr_end_date | 광고 종료일 |
| jr_total_amount | 결제 금액 |
