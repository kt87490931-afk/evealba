# 이브알바 쿠폰시스템 제작 플랜

## 1. 개요

기업회원 대상 쿠폰(줄광고3달무료, 채용공고할인, 썸네일옵션할인) 및 가입/월간 자동 발급 기능을 구현한다.

### 1.1 어드민 요구사항

- **쿠폰번호(ec_code) 미사용**: 어드민·사용자 화면에 노출하지 않음
- **기간별 발급 제어**: 발급 가능 기간, 쿠폰 사용 유효기간 설정
- **개인/일괄 발급**: 특정 회원 발급, 조건별 일괄 발급, 발급 내역 조회

---

## 2. 확인된 사항

### 2.1 기업회원 판별 기준

| 항목 | 값 | 근거 |
|------|-----|------|
| **판별 필드** | `mb_1` | g5_member 테이블 확장필드 |
| **기업회원 값** | `'biz'` | eve_register_update.php, eve_enterprise_list.php, ev_coupon.lib.php 등 일관 사용 |
| **승인 상태** | `mb_7` | pending(가입대기) / approved(승인) / rejected(반려) |
| **쿠폰 발급 대상** | `mb_1='biz' AND mb_7='approved'` | 승인된 기업회원만 쿠폰 사용·발급 |

### 2.2 관련 파일·테이블

| 구분 | 경로/테이블 |
|------|-------------|
| 회원가입 | `eve_register_update.php` |
| 기업회원 승인 | `adm/eve_enterprise_update.php` |
| 쿠폰 마스터 | `g5_ev_coupon` |
| 쿠폰 발급 | `g5_ev_coupon_issue` |
| 쿠폰 사용 로그 | `g5_ev_coupon_use` |
| 쿠폰 라이브러리 | `lib/ev_coupon.lib.php` |
| 썸네일 구매 | `jobs_thumb_option_purchase.php` (쿠폰 연동 있음) |
| 채용공고 결제 | `jobs_register_update.php`, `theme/evealba/jobs_register_main.php` (가격계산) |
| 광고 섹션 | `theme/evealba/inc/jobs_register_ad_section.inc.php` |

### 2.3 광고 금액 구조

- **줄광고** (할인 불가): 30일 70,000 / 60일 125,000 / 90일 170,000
- **박스광고** (채용공고 할인 적용): 우대, 프리미엄, 스페셜, 급구 등
- jr_ad_labels: "줄광고 30일, 우대 30일" 등 문자열로 저장

## 3. 쿠폰 종류 및 정책

### 3.1 쿠폰 유형

| 쿠폰명 | ec_type | 할인 방식 | 적용 대상 | 비고 |
|--------|---------|-----------|-----------|------|
| 줄광고 3달 무료 | `line_ad_free` | 90일 정가 전액 차감 | 줄광고 90일 1회 | 신규 타입 |
| 채용공고 10~50% 할인 | `ad` | percent | 박스광고 금액만 | 줄광고 제외 |
| 썸네일옵션 10~50% 할인 | `thumb` | percent | 썸네일옵션 | 기존 ec_type 유지 |

### 3.2 중복 적용 규칙

- 줄광고 + 채용공고 할인: **동시 적용 가능**
- 채용공고 할인은 **줄광고 금액에는 적용 안 함** (박스광고 금액만 할인)

### 3.3 자동 발급 정책

| 시점 | 쿠폰 | 수량 | 비고 |
|------|------|------|------|
| 기업회원 가입 시 | 줄광고3달무료, 채용공고30%할인 | 각 1장 | **발급 시점 선택 필요** (아래 4.1) |
| 매월 1일 | 지정 쿠폰 (예: 30%할인) | 각 1장 | 전체 승인 기업회원 대상 |

## 4. 발급 시점 (선택)

### A안: 가입 직후 발급 (eve_register_update.php)

- INSERT 성공 직후 g5_ev_coupon_issue에 INSERT
- 이때 mb_7='pending' → 로그인 불가이므로 쿠폰은 보유만 하고 승인 후 사용

### B안: 승인 시 발급 (adm/eve_enterprise_update.php)

- approve 처리 시 g5_ev_coupon_issue에 INSERT
- 승인된 회원만 쿠폰 보유 → 로직 단순, 추천

**권장: B안 (승인 시 발급)**

## 5. 스키마 확장

### 5.1 ec_type 확장

- 기존: `thumb`, `ad`, `gift`
- 추가: `line_ad_free` (줄광고 무료 전용)

### 5.2 ec_subtype 또는 ec_exclude (선택)

- 채용공고 할인 시 "줄광고 제외" 규칙을 ec_exclude='line_ad' 등으로 저장 가능
- 또는 코드에서 ec_type='ad' + 라벨 구분으로 처리 (추천)

### 5.3 쿠폰번호(ec_code) 미사용

- ec_code: DB에는 유지, UNIQUE 제약 충족을 위해 INSERT 시 자동 생성 (예: EV+ec_id 또는 UUID)
- 어드민·사용자 화면에 노출하지 않음

### 5.4 발급 기간 확장 (ec_issue_from, ec_issue_to)

- 마이그레이션 013 또는 012 보완으로 추가
- ec_issue_from, ec_issue_to: 이 기간에만 발급 가능 (NULL=제한없음)
- ec_issue_limit_per_member: 1인당 최대 발급 횟수 (0=무제한)

### Phase 1: 어드민 쿠폰 CRUD 및 발급 제어

**쿠폰번호(ec_code) 불필요**
- ec_code는 DB에는 유지하되, 내부 자동 생성(예: EV+ec_id)으로 UNIQUE 충족
- 어드민 UI에는 노출·입력 없음 (사용자 입력 X)

**최대한 제어 가능한 기능**

1. **기간별 발급**
   - **발급 가능 기간**: ec_issue_from, ec_issue_to (이 기간에만 발급 가능)
   - **쿠폰 사용 유효기간**: ec_valid_from, ec_valid_to (발급된 쿠폰의 사용 가능 기간)
   - 발급 시 유효기간: "발급일+N일" 또는 "고정 기간" 선택

2. **개인 발급**
   - **개별 발급**: 특정 회원(mb_id) 검색 후 1명에게 발급
   - **일괄 발급**: 조건(기업회원 전체/검색결과) 선택 후 다수 회원에게 발급
   - **발급 내역 조회**: ec_id별/회원별 발급·사용 내역

3. **발급 제어**
   - 1인당 발급 횟수 제한 (N회)
   - 총 발급 수 제한 (ec_use_limit)
   - 발급 즉시 / 예약(발급 가능 기간 내)

4. **adm/eve_coupon_list.php 확장**
   - 쿠폰 목록(코드 컬럼 미노출), 생성/수정/삭제, 발급 버튼
   - ec_type: thumb, ad, line_ad_free, gift
   - ec_valid_from, ec_valid_to, ec_issue_from, ec_issue_to (스키마 확장 시)
   - ec_discount_type, ec_discount_value, ec_min_amount, ec_max_discount

5. **adm/eve_coupon_form.php 신규**
   - 생성/수정 폼 (ec_code 입력란 없음)
   - 기간·제한 설정 UI

6. **adm/eve_coupon_form_update.php 신규**
   - POST 처리 (INSERT/UPDATE)
   - ec_code는 INSERT 시 자동 생성 (EV{ec_id} 또는 랜덤, UNIQUE 유지)

7. **adm/eve_coupon_issue.php 신규**
   - 개인 발급: 회원 검색 → 선택 → 발급
   - 일괄 발급: 조건(기업회원 등) 선택 → 발급 실행
   - 발급 내역 테이블 (회원ID, 쿠폰명, 발급일, 사용여부)

### Phase 2: 채용공고 결제 쿠폰 연동

1. **가격 계산 로직 분리**
   - `jobs_register_main.php` (또는 관련 JS/서버)에서:
     - 줄광고 금액 합계
     - 박스광고 금액 합계
   - jr_ad_labels 파싱으로 라벨별 금액 산출

2. **쿠폰 선택 UI**
   - 채용공고 결제 단계에서 ev_coupon_list_available_ad($mb_id, $box_ad_amount) 호출
   - ec_type IN ('ad','line_ad_free'), ec_target='biz' 대상

3. **할인 적용 규칙**
   - line_ad_free: 줄광고 90일 1개 포함 시 170,000원 차감 (1회 사용)
   - ad: 박스광고 금액에 percent 할인
   - 줄광고 금액은 항상 정가

4. **lib/ev_coupon.lib.php 확장**
   - `ev_coupon_list_available_ad($mb_id, $line_amount, $box_amount)` 추가
   - line_ad_free: line_amount >= 170000 시 사용 가능
   - ad: box_amount 기준

5. **jobs_register_update.php**
   - coupon_id, coupon_discount 수신
   - jr_total_amount 반영 (할인 적용된 최종 금액)

### Phase 4: 가입/승인 시 자동 발급

1. **발급 시점**: B안(승인 시) 권장
2. **adm/eve_enterprise_update.php 수정**
   - approve 분기에서:
     - ec_name 또는 설정된 ec_id로 조회 (예: '줄광고3달무료', '채용공고30%할인')
     - g5_ev_coupon_issue에 (ec_id, mb_id) INSERT

3. **초기 쿠폰 데이터**
   - adm 또는 시드 스크립트로 다음 쿠폰 등록:
     - 줄광고3달무료 (ec_type=line_ad_free, ec_discount_type=amount, ec_discount_value=170000)
     - 채용공고30%할인 (ec_type=ad, ec_discount_type=percent, ec_discount_value=30)
     - 채용공고 10/20/40/50% 할인 (선택)
     - 썸네일 10/20/30/40/50% 할인 (선택)

4. **유효기간**
   - 어드민에서 ec_valid_from, ec_valid_to 설정
   - 발급 시 eci_issued_at 기준 + N일 로 만료 처리할지는 정책 결정

### Phase 5: 월 1일 자동 발급 (Cron)

1. **cron_monthly_coupon_issue.php 신규**
   - 위치: `cron/cron_monthly_coupon_issue.php` 또는 `scripts/`
   - 로직:
     - 기업회원 조회: `SELECT mb_id FROM g5_member WHERE mb_1='biz' AND mb_7='approved'`
     - 지정 ec_id(예: 월별 30% 할인 쿠폰)에 대해 g5_ev_coupon_issue INSERT
     - 중복 발급 방지: 이미 해당 ec_id+mb_id 조합이 있으면 스킵 (또는 월별 1회 제한)

2. **Cron 등록**
   - `0 0 1 * *` (매월 1일 00:00)
   - `php /var/www/evealba/cron/cron_monthly_coupon_issue.php`

3. **어드민 월간 쿠폰 설정**
   - ec_id 중 "매월 1일 발급 대상" 표시 플래그(ec_auto_monthly) 또는 별도 설정 테이블
   - 또는 고정 ec_id를 스크립트에 하드코딩 후 어드민에서 해당 쿠폰만 수정

## 7. 동일 실수 시 해결책

| 상황 | 원인 | 해결 |
|------|------|------|
| g5_ev_coupon 없음 | 012 미실행 | run_migration_012.php 실행 |
| 쿠폰 조회 0건 | ec_target/ec_type 불일치 | ec_target='biz', ec_type IN ('ad','thumb','line_ad_free') 확인 |
| 기업회원 아님 | mb_1 불일치 | mb_1='biz' 확인 (business 아님) |
| 승인 전 쿠폰 사용 시도 | mb_7='pending' | 로그인 차단되어 접근 불가. 발급은 B안 시 승인 후만 |
| 줄광고에 채용 할인 적용됨 | 금액 분리 로직 오류 | 줄광고/박스광고 금액 분리 후 박스에만 할인 적용 |
| 중복 발급 | INSERT 전 체크 없음 | INSERT 전 `SELECT eci_id FROM g5_ev_coupon_issue WHERE ec_id=? AND mb_id=? AND ...` 체크 |
| 월 1일 미실행 | Cron 미등록/경로 오류 | crontab -e 확인, 절대경로 사용, php 경로 확인 |
